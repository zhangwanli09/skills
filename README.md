# skills

A [Claude Code](https://claude.ai/code) plugin — reusable skills, commands, and agents that extend Claude Code's capabilities.

## Skills

| Skill | Description |
|-------|-------------|
| [init-nextjs](./skills/init-nextjs/SKILL.md) | Initialize a new Next.js app with Prettier and pre-commit git hooks |

## Adding a New Skill

```bash
mkdir -p skills/my-new-skill
```

Create `skills/my-new-skill/SKILL.md` with frontmatter (`name`, `description`, `version`) and instructions. See [CLAUDE.md](./CLAUDE.md) for the full guide.
